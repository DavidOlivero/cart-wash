# Para iniciar el proyecto primero es necesario instalar bunjs:

- Para Windows:
```bash
powershell -c "irm bun.sh/install.ps1 | iex"
```

- Para Mac y Linux:
```bash
curl -fsSL https://bun.sh/install | bash
```

## Luego se debe instalar las dependencias:

```bash
bun install
```
